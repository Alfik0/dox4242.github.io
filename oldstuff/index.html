To play the last, old-style UI version of Realm Grinder (with the giant treasure chest),
right-click and save the following link: Realm Grinder v1.6.55.0
Then, download and install GameLoad.
Open the Realm Grinder swf file in GameLoad.

Note: new save formats will not be able to be imported; you will have to import an old save format.
Note 2: without the Kongregate HTML wrapper code, you won't have access to the purchasable gifts.
