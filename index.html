Royal Advisory Board

Welcome! This is a collection of tools for the incremental game Realm Grinder by Divine Games.

As of 2015-10-29, the 1.4.37 update has greatly increased the obfuscation of Realm grinder's code, leading to me having to read stuff like this:

if (4P.`@(0;.id)){

rather than stuff like this:

if (Game.hasUpgrade("Angelic Determination")){

for things like extracting formulas for the wiki, reporting bugs to Divine Games, and developing tools like these. This obfuscation has made these tasks much less pleasant, and with me being on the long slog to R16 when the update hit, spoiled my plan to kill time while I waited. I have asked Divine Games about this change and possible solutions, but as things are now, it is unlikely I will maintain Realm Weather Service, write any further tools, or continue editing the wiki. I will update this page if I get any further information.

At present and perhaps indefinitely, the only tool here is:

Realm Weather Service is a forecasting tool for Lightning Strike. It extracts the RNG state from a provided save file and predicts the future pattern of strikes.

Other Things

Everything here is open source and licensed MIT, here's the source code.

These tools use libraries by various lovely people, namely zlib.js and jQuery.

I have also written some simple libraries as part of this project, which you might find useful if you are a codey sort.

  • pmprng is a port of realm grinder's RNG with some extra helper functions for the various upgrades and spells with dedicated RNG instances, like lightning strike. There are python and javascript implementations.
  • savecodec is a library that converts between exported saves and objects/dicts. There are python and javascript implementations.
