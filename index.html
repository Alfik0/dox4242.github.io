Royal Advisory Board

Welcome! This is a collection of tools for the incremental game Realm Grinder by Divine Games.

Tools

Realm Weather Service is a forecasting tool for Lightning Strike. It extracts the RNG state from a provided save file and predicts the future pattern of strikes.

Forthcoming: Obsessive Conjurer Disorder will be a contingency autocasting threshold fine-tuner.

Forthcoming: Your Build on Drugs will be a random merc build generator of dubious utility.

Forthcoming: Royal Scribe will be a statistics utility that will extract a number of stats from save files that are tracked but not displayed in-game.

Conceptual: Bagger-288 will be an excavation reward forecaster similar to Realm Weather Service.

Conceptual: Lightning rod will be a lightning strike cheat that stacks the deck by loading in an RNG state that will produce a number of hits in a row on a desired target.
