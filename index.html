Royal Advisory Board

Welcome! This is a collection of tools for the incremental game Realm Grinder by Divine Games.

Tools

Realm Weather Service is a forecasting tool for Lightning Strike. It extracts the RNG state from a provided save file and predicts the future pattern of strikes.

Forthcoming: Obsessive Caster Disorder will be a contingency autocasting threshold fine-tuner.

Forthcoming: Royal Scribe will be a statistics utility that will extract a number of stats from save files that are tracked but not displayed in-game.
